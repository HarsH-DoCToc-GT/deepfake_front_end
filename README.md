# Deepfake Front End

## Project Description
This project is a front-end application designed to interact with deepfake detection systems. The goal is to provide a user-friendly interface for uploading videos and receiving feedback on their authenticity.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/HarsH-DoCToc-GT/deepfake_front_end.git
   ```
2. Navigate to the project directory:
   ```bash
   cd deepfake_front_end
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## How to Run the Application
To start the development server, run:
```bash
npm start
```
The application will then be available at `http://localhost:3000`. Modify the application as needed and refresh the browser to see the changes in real time.
